Rask::Task

Task base class

To define new Task you must inherit this base-class

  • sample code 
 class NewTask < Rask::Task
   define_state :initial, :initial => true
   define_state :finish
   def initial
     transition :finish
   end
   def finish
     destroy
   end
 end
